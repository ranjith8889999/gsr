## Image Guidelines:

- **Format**: JPG or PNG for images, MP4 for videos
- **Quality**: High resolution for best display
- **Size**: Optimize images for web (keep under 500KB if possible)
- **Aspect Ratios**:
  - Hero image: 16:9
  - Event cards: 3:2
  - Gallery images: Can vary, but maintain consistency
  - Profile photo: 4:5 (portrait)

## How to Add Your Images:

1. Replace the placeholder image names with your actual photos
2. Keep the exact filenames as listed above, OR
3. Update the image source paths in index.html to match your actual filenames

## Placeholder Images:

If you don't have all images ready, you can use placeholder services temporarily:
- https://via.placeholder.com/1200x800
- https://picsum.photos/1200/800

Example: `<img src="https://via.placeholder.com/1200x800" alt="Placeholder">`
